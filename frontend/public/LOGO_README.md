# Restaurant logo (e.g. pomegranate)

To use a **pomegranate** (or any image) as your restaurant logo:

1. **Add your image** to this folder (`frontend/public/`).
   - Example: save your pomegranate image as `logo.png` (or `logo-pomegranate.png`).

2. **Set the logo in Admin:**
   - Log in as Admin → **Settings**.
   - In **Logo URL** enter: `/logo.png` (or whatever you named the file, e.g. `/logo-pomegranate.png`).
   - Save.

3. The logo will appear in the **navbar** next to your restaurant name.

You can use any image (PNG, JPG, WebP). Keep it simple and not too large (e.g. under 200 KB) so the navbar loads quickly.
